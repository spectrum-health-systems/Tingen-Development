      Remarks

      This is the main entry point for the Tingen web service, and contains the two methods required by Avatar:

      • GetVersion
      • RunScript(OptionObject2015, String)

      This class does very little work itself (the heavy lifting is done in Outpost31) and should not be modified.

      Please see the Tingen documentation for more information.
